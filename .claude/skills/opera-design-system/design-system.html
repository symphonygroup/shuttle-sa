Quality checklist (this page)

Every item is satisfied by the markup on this page. If you change the system and any line breaks here, dark-mode toggling exposes it instantly.

  • All text uses Poppins (loaded above as 400/500/600/700).
  • Every font size matches the type ramp — no 17, 22, 36 outliers.
  • Every color references a CSS variable; no raw hex in component selectors.
  • Primary CTA fill is var(--color-bg-primary) = #6C69FF in both modes.
  • Status badges flip from 100/700/300 on light to alpha-derived on dark.
  • Wordmark swaps to white under .dark via <picture> + JS mirror.
  • Scrim uses --color-scrim, never hard-coded rgba.
  • Theme preference persists to localStorage as opera-theme.
  • SSR flash-prevention <script> runs before any stylesheet.
  • Focus ring (try Tab through the page) is always 2px #6C69FF at 2px offset.
